Informations du projet

  • Personnes: Moi-même, Ozcan Asci, Theo Foutren, Theo Lefevre
  • Client: Laurent Neel
  • Durée du projet : de Janvier à Juin 2022

Le projet Parking Partagé nécessite une plateforme de réservation pour les clients. J’ai donc réalisé une application Android de moi-même.

On a donc besoin d’applications permettant aux clients de créer un compte pour réserver une place de parking et obtenir leur code d’accès. Elles seront accessibles via l’application directe pour les utilisateurs Android et par navigateur Internet pour les utilisateurs provenant d’autres plateformes.
L’application doit donc pouvoir lui attribuer la place puis lui fournir le code d’accès au parking.

C'est ce que j'ai réalisé.
